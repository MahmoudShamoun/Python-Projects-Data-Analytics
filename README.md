## 1️⃣ Netflix Content Analysis

**Description:** A comprehensive end-to-end data analysis project on the Netflix titles dataset. The project covers the full data science workflow — from raw data ingestion to publishing publication-quality charts styled with the official Netflix color palette.

**Key Features:**
* **Full Data Cleaning Pipeline:** Systematic handling of missing values and whitespace stripping.
* **Feature Engineering:** Extracting temporal insights (Date/Year added) and numeric components from string durations.
* **Content Strategy Analysis:** Detailed split between Movies and TV Shows with production trends over time.
* **Global & Genre Insights:** Identifying top producing countries and dominant genres using data explosion techniques.
* **Custom Visualizations:** 10 themed plots including monthly heatmaps, histograms, and trend lines using the Netflix palette (`#E50914`).

**Skills Used:** Python, Pandas, NumPy, Matplotlib, Seaborn.

**Purpose:** Demonstrates the ability to transform raw datasets into strategic business insights and professional visual storytelling.
